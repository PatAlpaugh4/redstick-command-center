# Keyboard Shortcuts

> Complete reference for keyboard navigation and shortcuts in Redstick Ventures Dashboard.

## Global Shortcuts

These shortcuts work anywhere in the application:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl + K` | Open global search | Quickly search across deals, companies, and contacts |
| `?` | Show keyboard help | Display a modal with all available shortcuts |
| `Esc` | Close modal/dropdown | Close the currently open modal or dropdown |
| `/` | Focus search | Jump to the main search input |
| `Shift + ?` | Toggle shortcuts help | Quick reference overlay |

### Navigation Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Tab` | Next element | Move focus to the next interactive element |
| `Shift + Tab` | Previous element | Move focus to the previous interactive element |
| `Home` | First element | Jump to first item in a list |
| `End` | Last element | Jump to last item in a list |

## Custom Shortcuts

Users can customize certain shortcuts in **Settings > Keyboard Shortcuts**.

### Customizable Actions

- Global search shortcut
- New deal shortcut
- Navigation shortcuts
- Export shortcut

### Conflict Resolution

If a custom shortcut conflicts with:
- **Browser shortcuts**: Application shortcut takes precedence with warning
- **System shortcuts**: System shortcut takes precedence
- **Other app shortcuts**: User can override with confirmation

## Learning Shortcuts

### Progressive Disclosure

New users see a **Shortcuts Tour** on first login:

1. **Day 1**: Global shortcuts only (Ctrl+K, ?, Esc)
2. **Day 3**: Context-specific shortcuts introduced
3. **Day 7**: Full shortcuts reference available

### Practice Mode

Access **Settings > Keyboard Shortcuts > Practice Mode** to:
- See shortcut hints overlaid on UI
- Practice shortcuts with guided exercises
- View your shortcut usage statistics

## Troubleshooting

### Shortcuts Not Working

1. Check if focus is in a text input (shortcuts may be disabled)
2. Verify no browser extensions are intercepting keys
3. Check **Settings > Keyboard Shortcuts** for custom bindings
4. Ensure keyboard layout matches your physical keyboard

### Reporting Issues

If you encounter shortcut conflicts or issues:
- Email: accessibility@redstick.vc
- GitHub: Create issue with `shortcuts` label
