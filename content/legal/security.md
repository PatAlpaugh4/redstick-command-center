# Security Overview

Last Updated: March 28, 2026

## Our Commitment to Security

At Redstick Ventures, security is foundational to everything we build. We implement industry-leading security practices to protect your data and maintain the trust you place in us.

## Security Certifications and Compliance

- SOC 2 Type II Certified
- GDPR Compliant
- CCPA Compliant
- Regular third-party security audits

## Data Protection

### Encryption
- **In Transit**: All data transmitted between your browser and our servers uses TLS 1.3 encryption
- **At Rest**: Data is encrypted using AES-256 encryption
- **Database**: Encrypted database storage with managed keys

### Access Controls
- Role-based access control (RBAC)
- Multi-factor authentication (MFA) support
- Single Sign-On (SSO) integration
- Session management and timeout controls
- IP allowlisting (Enterprise plans)

## Infrastructure Security

### Cloud Infrastructure
- Hosted on secure cloud providers (AWS/GCP)
- Network isolation and VPC configuration
- DDoS protection
- Web Application Firewall (WAF)

### Monitoring and Logging
- 24/7 security monitoring
- Intrusion detection systems
- Comprehensive audit logging
- Automated threat detection
- Regular vulnerability scanning

## Application Security

### Secure Development
- Secure Software Development Lifecycle (SSDLC)
- Regular code reviews
- Static and dynamic application security testing
- Dependency vulnerability scanning
- Penetration testing (quarterly)

### Authentication
- bcrypt password hashing
- JWT token-based authentication
- Secure session management
- Automatic session timeout
- Password strength requirements

## Data Handling

### Data Retention
- Configurable retention policies
- Secure data deletion procedures
- Backup encryption and retention

### Data Access
- Principle of least privilege
- Regular access reviews
- Privileged access monitoring

## Incident Response

We maintain a comprehensive incident response plan:
- 24/7 incident response team
- Defined escalation procedures
- Communication protocols
- Regular incident response drills

## Business Continuity

### Backup and Recovery
- Automated daily backups
- Geographic redundancy
- Tested recovery procedures
- RPO: 24 hours, RTO: 4 hours

### Disaster Recovery
- Multi-region failover capability
- Documented disaster recovery procedures
- Regular DR testing

## Vendor Security

We carefully evaluate all third-party vendors:
- Security questionnaires
- SOC 2 or equivalent certification requirements
- Regular vendor security assessments
- Data processing agreements

## Your Security Responsibilities

To maintain security, you should:
- Use strong, unique passwords
- Enable multi-factor authentication
- Keep your devices secure
- Report suspicious activity immediately
- Follow your organization's security policies
