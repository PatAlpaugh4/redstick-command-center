## Part 1: VC Website References

### 1.1 Sequoia Capital (sequoiacap.com)
**Design Archetype:** Dynamic, Motion-Forward

**Key Elements:**
- Animated "marker pen" text effect on headlines
- Colorful tile-based content organization
- Motion toggle for performance (On/Off)
- Bold, punchy headlines
- Interactive portfolio filtering
- Animated GIFs for engagement

**Implementation Notes:**
- Use Framer Motion for text reveal animations
- Implement scroll-triggered tile reveals
- Add motion preference toggle
- Colorful accent tiles on dark background

## Part 7: Design Patterns Summary

### Color Patterns
| Pattern | Usage | Example |
|---------|-------|---------|
| Dark background | Primary theme | #0f0f1a |
| Navy surface | Cards, panels | #1a1a2e |
| Red accent | CTAs, active states | #e94560 |
| Green | Success, active agents | #4ade80 |
| Amber | Warning, paused | #fbbf24 |

### Layout Patterns
| Pattern | Usage | Source |
|---------|-------|--------|
| Sidebar + Content | Command center | Mission Control |
| Card grid | Agent display | AgentCenter |
| Kanban board | Pipeline | Linear, Trello |
| Node diagram | Orchestration | Kubiya |

### Animation Patterns
| Pattern | Usage | Implementation |
|---------|-------|----------------|
| Text reveal | Hero headlines | Framer Motion |
| Card hover | Interactive elements | CSS transitions |
| Status pulse | Live indicators | CSS animation |
| Data flow | Orchestration | SVG + GSAP |
| Scroll reveal | Section entry | Intersection Observer |
