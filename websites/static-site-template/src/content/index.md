# Welcome to Static Site Template

This is a simple, fast static site generator pattern. Write your content in **Markdown** and let the build script do the rest.

## Features

- ✨ Markdown support
- 🎨 Template partials
- 📦 Zero runtime dependencies
- 🚀 Fast builds

## Getting Started

1. Edit content in `src/content/`
2. Run `npm run build`
3. Deploy the `dist/` folder
